# Text

## Use

- Handle text styling, emphasis, links, footnotes, raw text, superscripts, subscripts, and inline semantic text elements.

## Authoritative Sources

- `typst/docs/reference/library/text.md`
- `typst/crates/typst-library/src/text/**`

## Key Objects

- `text`, `raw`, `linebreak`
- `emph`, `strong`, `highlight`, `smallcaps`
- `sub`, `super`, `underline`, `overline`, `strike`
- `link`, `footnote`, `smartquote`

## Debug Tips

- Separate text-element issues from layout or model-structure issues.
- In `0.14.x`, many text-like elements also matter for export semantics and accessibility.
