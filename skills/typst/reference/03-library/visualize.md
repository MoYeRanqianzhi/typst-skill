# Visualize

## Use

- Handle color, stroke, gradient, shape, image, and decoration-related Typst tasks.

## Authoritative Sources

- `typst/docs/reference/library/visualize.md`
- `typst/crates/typst-library/src/visualize/**`
- `typst/docs/changelog/0.14.0.md`

## Key Objects

- Color and line styling: `color`, `stroke`, `gradient`, `tiling`
- Shapes: `rect`, `square`, `circle`, `ellipse`, `polygon`, `curve`, `line`
- Images: `image`

## Version Notes

- `0.14.0` added support for using PDFs as images.
- `0.14.0` added WebP support.
- Decorative graphics should often be treated as artifacts or paired with `figure.alt` when they carry meaning.
