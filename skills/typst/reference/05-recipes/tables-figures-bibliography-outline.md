# Tables, Figures, Bibliography, Outline

## Use

- Handle common report, paper, and book composition tasks.

## Key Patterns

- Tables: stabilize column logic and headers before styling.
- Figures: prefer `figure`; add `figure.alt` when accessibility matters.
- Bibliography: centralize citation style and source handling in templates.
- Outline: stabilize heading levels and numbering before generating outlines.
