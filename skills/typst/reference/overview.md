# Overview

## Use

- Open this file when you need the shortest possible route map before choosing a deeper reference page.

## Authority Order

1. local `typst/` source and official docs
2. generated indexes under `reference/generated/` and `reference/08-generated/`
3. local blue-book examples

## Quick Routes

- write or fix Typst code: open the smallest matching page under `01-workflows/`, `02-language/`, `03-library/`, or `05-recipes/`
- exact API lookup: run `python skills/typst/scripts/query_reference.py --query <keyword>`
- fast official inventory lookup: run `python skills/typst/scripts/query_api_index.py --name <keyword>`
- source-level maintenance: open `06-dev/`
- version disputes or drift: open `07-versioning/`
